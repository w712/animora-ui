# Animora UI

> 动画驱动的 Vue 组件库，让每一次交互都充满生命力

## 快速开始

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建
pnpm build
```

## 项目结构

```
animora-ui/
├── packages/
│   └── animora-ui/    # 核心组件库
├── docs/              # 文档站点
├── playground/        # 演示项目
└── examples/          # 示例项目
```

## 技术栈

- Vue 2 / Vue 3 (vue-demi)
- TypeScript
- GSAP (可选)
- Vite
- VitePress
