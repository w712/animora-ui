# 组件开发记录 - 2026-01-02

## 新增组件

### 1. ACard 卡片组件

位置：`packages/animora-ui/src/components/card/`

动画效果：

- `hover-lift` - 悬停上浮
- `hover-glow` - 悬停发光
- `tilt` - 3D 倾斜跟随鼠标

```vue
<ACard title="标题" animation="hover-lift">
  内容
</ACard>
```

### 2. ADialog 对话框组件

位置：`packages/animora-ui/src/components/dialog/`

动画效果：

- `zoom` - 缩放
- `fade` - 淡入淡出
- `slide-down` - 从上滑入
- `slide-up` - 从下滑入
- `flip` - 翻转
- `bounce` - 弹跳

```vue
<ADialog v-model="visible" animation="zoom" title="标题">
  内容
</ADialog>
```

### 3. ATabs 标签页组件

位置：`packages/animora-ui/src/components/tabs/`

动画效果：

- `slide` - 滑动
- `fade` - 淡入淡出
- `scale` - 缩放
- `flip` - 翻转

```vue
<ATabs v-model="activeTab" animation="slide">
  <ATabPane label="首页" name="tab1">内容1</ATabPane>
  <ATabPane label="用户" name="tab2">内容2</ATabPane>
</ATabs>
```
