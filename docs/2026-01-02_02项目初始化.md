## Phase 2 - 动画系统 ✅

完成时间: 2026-01-02

### 完成内容

- [x] 动画引擎 (core/engine.ts)
  - 自动检测 GSAP
  - CSS Animation 降级
  - 统一 API

- [x] 预设动画 (38个)
  - Fade (6): fadeIn, fadeOut, fadeInUp, fadeInDown, fadeInLeft, fadeInRight
  - Slide (8): slideInUp, slideInDown, slideInLeft, slideInRight, slideOutUp, slideOutDown, slideOutLeft, slideOutRight
  - Zoom (4): zoomIn, zoomOut, zoomInUp, zoomInDown
  - Flip (4): flipInX, flipInY, flipOutX, flipOutY

  - 支持 .click 修饰符
  - 支持 .scroll 修饰符

- [x] useAnimate Composable
  - animate() 方法
